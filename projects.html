NLP-Suite

Supervisor: Roberto Franzosi

A freeware, open-source toolkit. The goal is providing humanists and social scientists a wide range of computational tools for the analysis and visualization of smaller datasets. It is designed for non-specialists, for scholars with no knowledge or little knowledge of Natural Language Processing.

gtars

Supervisor: Nathan Sheffield

A rust crate that provides a set of tools for working with genomic interval data. Its primary goal is to provide processors for geniml.

geniml

Supervisor: Nathan Sheffield

A python package for building machine learning models of genomic interval data (BED files). It also includes ancillary functions to support other types of analyses of genomic interval data.

acoustic-solo-dadaGP

A modification of dadaGP—a package that converts guitar tabular files to/from token format for sequence models—adapted for for acoustic guitar solos. The goal is support for different string tuning (like Celtic tuning) and process tokens based on the need of downstream training tasks.
